### find_sqli_url

######1、下载sqlmap。
######2、启动sqlmapapi服务，python sqlmapapi.py -s -H 127.0.0.1 -p 9090
######3、下载find_sqli_url.py，输入搜索关键字，列如inurl:news.aspx?id=
######4、然后输入你想搜索的条数。
######5、等待收鱼吧。
######6、也可以将url通过其他方式存放在文本文档中，然后使用from_urlfile_scan.py进行批量注入。

######tips:第三方库麻烦自己用pip install xxxxx装一下咯。
######成功率取决于关键字~!~
